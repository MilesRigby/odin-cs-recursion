# odin-cs-recursion
JavaScript implementations for outputting a series of Fibonacci numbers and performing the merge sort algorithm on an array of numbers

Lesson on the Odin Project's site suggests using nodejs to allow functions to be used in the command line. As at the time of me completing this npm is compromised by the Shai-Hulud V2 Worm, I'm still using an index.html file to load the JS into a browser console.
